# mermaid example

ライブラリ読み込み

```js echo
import mermaid from "npm:@observablehq/mermaid";
```

## mermaidを使ってフロー図を書く
<pre>
```mermaid 
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
